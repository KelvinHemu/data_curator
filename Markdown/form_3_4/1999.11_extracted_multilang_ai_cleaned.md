## SECTION A (56 marks)
Answer ALL questions in this section. ALL working for each question must be shown clearly.

1.  (a) Evaluate $\int_0^2 (4 - x) \sin(\frac{\pi x}{4}) dx$ by substitution method. (4 marks)

    (b) Find the equation of a curve which passes through the point (1,0) whose gradient function is $\frac{3x-1}{2}$. (3 marks)

2.  (a) Find the inverse of the matrix
    $A = \begin{bmatrix} 1 & 2 \\ 2 & -1 \end{bmatrix}$ (3 marks)

    (b) Solve the following system of equations by using the inverse of matrix A obtained in (a) above:
    $2y + x = 10$
    $2x - y = 5$

3.  (a) Write $\frac{3-i}{4-3i}$ in standard form. (2 marks)

    (b) Given that $z = i(3 + 4i)$, find $|z|$. (3 marks)

    (c) Express $1 + i$ in polar form. (2 marks)

4.  (a) If $n(A) = 60$, $n(B) = 50$, $n(C) = 18$, $n(A \cap B) = 30$, $n(B \cap C) = 10$, $n(A \cap C) = 6$, $n(A \cap B \cap C) = 4$, find $n(A \cup B \cup C)$. (3 marks)

    (b) In a class of 34 girls, 21 play tennis and 18 play netball. If all the girls play at least one of these games, how many play both netball and tennis? (4 marks)

5.  (a) If $y = \frac{1 + \sin\theta}{\cos\theta}$, show that $\frac{1}{y} = \frac{1 - \sin\theta}{\cos\theta}$. (3 marks)

    (b) If $s = \sin\theta$ and $c = \cos\theta$, simplify:

    (i) $\frac{s}{\sqrt{1-s^2}}$
    (ii) $\frac{\sqrt{1-s^2}}{c}$ (4 marks)

6.  (a) Given that
    $f(x) = \begin{cases}
    -2 & \text{for } x > 1 \\
    1 & \text{for } x = 1 \\
    3 & \text{for } x < 1
    \end{cases}$

    (i) Sketch the graph of this function.
    (ii) Evaluate $f(-5)$. (4 marks)

    (b) Solve for $t$, $|3-2t| < 5$. (3 marks)

7.  (a) Find the values of $a$ and $b$ if the expression $x^4 + ax^3 + bx^2 - 4$ is exactly divisible by $x^2 - 4$. (4 marks)

    (b) Determine the factors of the expression $x^2 - 2x - 3$. (3 marks)

8.  (a) The probability that Halima will pass Mathematics is 0.4 and that Perpetua will do the same is 0.7. Find the probability that:

    (i) both will pass Mathematics
    (ii) Halima or Perpetua will pass Mathematics. (4 marks)

    (b) Find $x$ in $\binom{n}{x} = 3$ when $\binom{n}{2} = 36$. (3 marks)

If chair fetches Tshs.400 as profit and a table Tshs. 400/=, determine the best combination of tables and chairs to produce so as to get maximum profit. (11 marks)
